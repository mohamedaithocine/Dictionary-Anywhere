# Dictionary Anywhere (DictionaryAPI)
The **Dictionary Anywhere** extension helps you stay focused on what you are reading by eliminating the need to search for meaning, 
Double-clicking any word will view its definition in a small pop-up bubble. 
Now you never have to leave what you are reading to search for the meaning of the words you don't yet know.

## Installation
Clone the repository or download it as a zip file <br>
Unzip the folder <br>
In Firefox go to Manage Extensions or `about:addons` <br>
Click the cog > Install add-on from File... <br>
Select Dictionary-Anywhere-master.xpi <br>
Click Add <br>

### Solution: Add-on could not be installed because it is not verified.
Go to `about:config` <br>
Set `xpinstall.signatures.required` to false <br>
Try install add-on from file again from `about:addons` <br>
Click Add <br>
